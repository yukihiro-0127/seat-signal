# 📍 Seat Signal

箱崎事業所（17F〜25F）向けの空席可視化アプリ MVP

## 概要

社員が匿名でエリアの利用状況を共有し、フロア・エリアごとの空席状況をリアルタイムで可視化するツールです。  
「座席予約システム」ではなく、「席探しの探索コストを下げる」ことを目的としています。

## 主な機能

| 画面 | 内容 |
|------|------|
| **Home** | 全体混雑度・TOP推薦エリア・フロア別サマリー・貢献ポイント |
| **Map** | 17F〜25F 全9フロア × 4エリア = 36エリアの空席状況 |
| **Check In** | 3ステップで匿名チェックイン・混雑報告・チェックアウト |
| **Find** | 自然文 or クイックチップ入力でAI風おすすめエリア推薦 |
| **Points** | 貢献ポイント残高・獲得ルール・履歴 |

## ポイントルール

- Check In: **+5 pts**
- Check Out: **+5 pts**
- Leaving Soon 共有: **+3 pts**
- 混雑報告: **+3 pts**

## プライバシー設計

- 個人名・メールアドレス・社員番号は一切使用しない
- エリア単位の匿名集計のみ
- 個人の座席位置は追跡しない

## 技術スタック

- **フロントエンド**: HTML / CSS / Vanilla JavaScript（単一ファイル）
- **バックエンド**: なし（デモ用インメモリデータ）
- **デプロイ**: `index.html` をブラウザで開くだけで動作

## 使い方

```bash
# ブラウザで直接開く
open index.html
```

GitHub Pages でそのままホスティングも可能です。

## Impact 試算

> 1人あたり1日3分の席探し時間を削減できた場合  
> 100人 × 3分 × 20営業日 = **月100時間の削減余地**
